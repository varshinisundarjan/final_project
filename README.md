# Online Course Exam System

## Repository Name
github_final_project

## Project Name
Online Course Exam System (Django Web Application)

## Project Description
This project is a Django-based web application developed to manage online course examinations. 
The system allows instructors to create courses, add exam questions, and evaluate student submissions.

Students can log in, take exams, submit answers, and view their results. The application uses Django models, views, and templates to manage course content and exam evaluation.

## Technologies Used
- Python
- Django Framework
- HTML, CSS, Bootstrap
- SQLite / MySQL Database

## Features
- User authentication (student login)
- Course and exam management
- Question and choice creation
- Automatic exam result calculation
- Admin panel for managing courses and questions
